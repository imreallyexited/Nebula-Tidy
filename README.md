**Nebula-Tidy** is a basic-standalone desktop file organizer. It organizes your desktop files into folders automatically.

## Features

* **Standalone EXE:** No installation required, no external folders.
* **System Guard:** Checks system health (RAM/Disk) before operations.
* **Auto-Refresh:** Automatically detects new files on the desktop.
* **Smart Selection:** Filter specific file types to organize.

## Installation & Usage

1.  Download `Nebula-Tidy.exe` from the [Releases](link-to-releases) section.
2.  Type a folder name (e.g., "MyFiles").
3.  Select the file types you want to move.
4.  Click **MOVE FILES**.
